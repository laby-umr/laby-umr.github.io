##  学习指南

**核心要点：**
- Spring Boot 自动配置原理
- Starter 机制和自定义Starter
- 配置文件和环境管理
- Spring Boot 在微服务中的应用

**学习路径建议：**
1. 掌握 Spring Boot 自动配置机制
2. 熟悉常用Starter的使用
3. 学习配置文件和环境管理
4. 理解Spring Boot在微服务架构中的应用
