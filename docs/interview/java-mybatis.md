##  学习指南

**核心要点：**
- MyBatis 的SQL映射和执行流程
- 一级缓存和二级缓存机制
- 动态SQL和插件机制
- MyBatis-Plus扩展功能

**学习路径建议：**
1. 掌握MyBatis的基本配置和使用
2. 理解缓存机制和性能优化
3. 熟悉动态SQL和插件开发
4. 学习MyBatis-Plus的实用功能
